# 회원가입 페이지

회원가입 페이지의 주요 동작을 문서화합니다.

## 📌 기능 목록

- **닉네임 입력 테스트**: 닉네임을 입력할 수 있다.
- **자기소개 입력 테스트**: 자기소개를 입력할 수 있다.
- **필수 입력 사항 체크**: 모든 필수 입력 필드가 입력되지 않으면 버튼이 비활성화된다.
- **회원가입 완료 버튼 동작**: 버튼을 클릭하면 이전 페이지로 이동한다.
- **닉네임 중복 확인 버튼 테스트**: 중복 확인을 할 수 있다.
