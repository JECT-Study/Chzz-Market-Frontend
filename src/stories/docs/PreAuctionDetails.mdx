# 사전 경매 상세 페이지

사전 경매 상세 페이지의 기능을 설명합니다.

## 📌 주요 기능

- 뒤로 가기 버튼 클릭 시 이전 페이지로 이동
- 최소 1장 이상의 이미지, 판매자 정보, 상품 이름, 카테고리, 시작가, 수정 시간, 찜한 사람 수 표시됨
- 카테고리 클릭 시 해당 카테고리 페이지로 이동
- 이미지 2장 이상일 경우 버튼으로 이미지 이동 가능
- 등록자만 옵션 버튼 사용 가능 (수정 및 삭제 가능)
- 등록자는 경매 전환하기 버튼을 사용하여 경매를 전환할 수 있음
