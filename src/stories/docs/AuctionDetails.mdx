# Auction Details

경매 상세 페이지에 대한 문서입니다.

## 📌 기본 기능

- 뒤로 가기 버튼을 클릭하면 이전 페이지로 이동한다.
- 최소 1장 이상의 이미지, 시간, 판매자 이름 및 프로필 사진, 상품 이름, 카테고리, 시작가, 참여금액, 참여 인원이 표시된다.
- 남은 시간이 1시간 미만이면 **빨간색**, 16시간 미만이면 **초록색**, 24시간 이하는 **파란색**으로 표시된다.
- 카테고리를 클릭하면 해당 카테고리의 경매 목록 페이지로 이동한다.

## 🛠 진행 중인 경매

- 판매자는 경매에 참여할 수 없다.
- 구매자가 경매에 참여하지 않은 경우 **참여하기 버튼**만 보이고, 클릭하면 입찰 페이지로 이동한다.
- 구매자가 경매에 참여한 경우:
  - **참여 취소 버튼**과 **금액 수정 버튼**이 보인다.
  - 참여 취소 버튼 클릭 시 **확인 모달**이 나타나며, 취소 시 해당 경매에 다시 참여할 수 없다.

## 🔚 종료된 경매

- 판매자는 **참여자 내역 보기 버튼**이 있으며, 참여자가 있을 경우 클릭하면 참여자 내역 페이지로 이동한다.
- 구매자가 참여하지 않았거나 낙찰받지 못한 경우, **종료된 경매 버튼**이 비활성화된다.
- 낙찰받은 경우 **결제하기 버튼**이 나타나며 클릭 시 결제 페이지로 이동한다.
- 결제가 완료된 경우 **결제 내역 버튼**이 활성화된다.
